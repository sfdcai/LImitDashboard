## Core Capabilities

### 1. Org Limits & Usage
Track daily API consumption, streaming usage, and storage profile snapshots to support forecasting and capacity planning.

### 2. License Utilization
Monitor used vs total licenses and identify under-utilized license pools.

### 3. Async Apex Reliability
Capture failed and in-flight jobs with error counts and processing details.

### 4. Integration Health
Log endpoint calls, response times, and failures for SLA reporting.

### 5. Email-to-Case Metrics
Measure attachment volume, email counts, and processing trends per case.

### 6. Data Retention Controls
Automatically purge old telemetry records to keep storage costs bounded.
